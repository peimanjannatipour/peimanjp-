# Peiman Jannatipour Portfolio

Professional researcher/inventor portfolio for Peiman Jannatipour / Peiman JP.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Vercel-compatible static/server rendering

## Commands

```bash
npm run dev
npm run lint
npm run build
```

## Content

Public copy is data-driven from `src/data`. Missing identifiers and downloads are hidden until verified or added.
